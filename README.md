# Restaurant Services and Pricing Analysis

This project focuses on analyzing restaurant data to understand how pricing is related to services such as online delivery and table booking.

## Objective
To study the relationship between restaurant pricing and the availability of services, and identify patterns that can help in understanding customer preferences and business strategies.

## Tools & Technologies
- Python  
- Pandas  
- Matplotlib  

## Work Performed
- Cleaned and preprocessed the restaurant dataset  
- Performed Exploratory Data Analysis (EDA)  
- Analyzed pricing trends with respect to online delivery and table booking services  
- Created visualizations to present insights clearly  

## Key Insights
- Identified pricing differences based on service availability  
- Observed trends that highlight how additional services impact restaurant pricing  

## Project Type
Individual Data Analysis Project

## Outcome
The project provides data-driven insights into restaurant pricing behavior, helping understand how services influence cost structures and customer offerings.
